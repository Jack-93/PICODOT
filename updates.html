서비스 업데이트

PICO DOT이 무엇을 바꾸고 있는지 날짜별로 기록합니다. 기능뿐 아니라 개인정보 처리, 광고와 운영 방식에 중요한 변경이 생기면 이 페이지에서 안내합니다.

2026년 6월 15일 v1.4

사진 기반 픽셀 아트 만들기 추가

  • 고양이 사진의 대표 털색, 밝은 털색과 어두운 색을 브라우저에서 분석합니다.
  • 분석한 색상을 정면 픽셀 고양이 템플릿에 자동으로 적용합니다.
  • 색상, 무늬와 표정을 직접 수정하고 1200px PNG로 저장할 수 있습니다.
  • 기능 메뉴를 픽셀 아트 만들기, 캐릭터 만들기와 사진 도트화로 구분했습니다.
  • 분석할 사진과 결과 이미지는 서버로 전송하거나 저장하지 않습니다.
2026년 6월 15일 v1.3

PICODOT 캐릭터 스튜디오 추가

  • 고양이 얼굴, 귀, 눈, 표정, 무늬와 액세서리를 조합할 수 있습니다.
  • 털, 무늬, 눈, 포인트와 배경 색상을 자유롭게 선택할 수 있습니다.
  • 랜덤 조합, 설정 자동 저장과 1200px PNG 다운로드를 지원합니다.
  • 모든 조립과 이미지 생성은 현재 브라우저 안에서 처리됩니다.
2026년 6월 15일 v1.2

팔레트, 디더링과 다운로드 크기 추가

  • 원본 기반, PICO 컬러, 게임보이, 흑백 4색과 파스텔 팔레트를 제공합니다.
  • 색상 사이의 단계를 점 패턴으로 표현하는 디더링 기능을 추가했습니다.
  • PNG 다운로드 크기를 긴 변 기준 600px, 1200px 또는 2400px로 선택할 수 있습니다.
  • 세로로 긴 사진도 처리량이 지나치게 커지지 않도록 도트 계산 방식을 안정화했습니다.
  • 팔레트, 디더링과 다운로드 크기도 현재 브라우저에 자동 저장합니다.
2026년 6월 15일 v1.1

비교 보기와 빠른 프리셋 추가

  • 도트 결과와 원본 사진을 같은 영역에서 전환해 비교할 수 있습니다.
  • 레트로, 균형, 디테일 세 가지 빠른 프리셋을 추가했습니다.
  • 도트 크기, 색상 수와 색감 설정을 현재 브라우저에 자동 저장합니다.
  • 다시 방문하면 마지막으로 사용한 설정을 자동으로 복원합니다.
  • 초기화 버튼은 사진과 저장된 설정을 함께 기본 상태로 되돌립니다.
  • 브라우저 설정 저장 방식과 삭제 방법을 개인정보처리방침에 반영했습니다.
2026년 6월 15일 v1.0.1

안정성과 모바일 사용 경험 개선

  • 선택한 사진의 파일명, 용량과 해상도를 변환 화면에서 확인할 수 있게 했습니다.
  • 설정 슬라이더를 빠르게 움직일 때 중복 렌더링을 줄여 반응성을 개선했습니다.
  • 가로·세로 사진 모두 긴 변을 기준으로 최대 1200px PNG가 생성되도록 정리했습니다.
  • 지원하지 않는 파일과 지나치게 큰 이미지에 대한 안내를 더 명확하게 표시합니다.
  • 모바일에서 사진 선택 버튼과 설정 슬라이더를 더 편하게 누를 수 있도록 조정했습니다.
  • 인물, 반려동물, 사물 사진별 추천 시작 설정과 v1 테스트 기준을 추가했습니다.
2026년 6월 13일 v1.0

색상 표현과 사진 선택 경험 개선

  • 사진 선택과 초기화 버튼을 분리해 조작을 더 명확하게 만들었습니다.
  • 권장 이미지 크기를 1200 × 1200px로 안내하도록 개선했습니다.
  • 색상 수 범위를 2색부터 64색까지 넓히고 실제 대표 색상 팔레트를 적용했습니다.
  • 색감 설정을 단조로운 표현부터 다양한 표현까지 조절할 수 있게 정리했습니다.
  • 사용하지 않는 격자 질감 기능을 제거했습니다.
2026년 6월 13일 콘텐츠·운영

공식 도메인과 이용 안내 정비

  • 공식 서비스 주소 picodot.net을 연결했습니다.
  • 사진 선택, 설정값, 결과물 활용과 저작권에 관한 가이드를 추가했습니다.
  • 개인정보처리방침과 운영자 문의 정보를 쉽게 확인할 수 있도록 정리했습니다.
  • AdSense 승인 전에는 빈 광고 영역이 화면에 표시되지 않도록 변경했습니다.
